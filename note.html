圆柱的体积

      定义：物体所占空间的大小叫物体的体积

      计算方法：

            圆柱的体积=底面积*高

            高=圆柱的体积/底面积

            底面积=圆柱的体积/高

            若将圆柱的底面分成若干相等的扇形：

分的扇形越多，就越接近长方形。

分出来后，

长方形的底面积=圆柱的底面积

高            =圆柱的高

比例的性质

总结：

      在比例中，两个外项的积等于两个内项的积。这叫做比例的基本性质

           

正片开始

      三三承包了两块农田，面积分别为0.5公顷和0.8公顷，秋收时，两块水稻田的产量分别为3.75t和6t

(1)   两块水稻田的面积和产量之比，是否可以组成比例

(2)   若可以，请指出比例的内项和外项

先写比：3.75 : 0.5       6 : 0.8

判断：3.75*0.8=3       0.5*6=3            可以

组成比例：3.75 : 0.5 =6 : 0.8

解比例

      啥是解比例？

            根据比例的基本性质，如果已知比例中的任何三项，就可以求出这个比例中的另外一个未知项。求比例中的未知项，叫做求比例

      正片开始

法国巴黎的埃菲尔铁塔高320m，北京世界公园内有一座埃菲尔铁塔的模型，它的高度与原塔的高度的比是1 : 10.这座塔高多少米？

                  解：设这座塔模型的高度为x m。

                        X : 320 = 1:10

                           10x = 320*1

                             X=320*1/10

                              X=32

                        或：x/320 = 1/10

                                  X = 1/10*320

                        答：高32m

            在一般情况下，我们一般把比例尺的前项（后项）写成1
